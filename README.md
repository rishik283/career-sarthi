Career Sarthi is an AI-powered career guidance web application designed to help users make informed career decisions. It provides personalized career suggestions, skill recommendations, and a structured roadmap based on user interests and goals.

Features
🔍 Personalized Career Recommendations
📊 Skill Gap Analysis
🧠 AI-based Career Suggestions
📚 Learning Roadmap Generator
💬 Interactive Chatbot Support
🔐 User Authentication (Login/Signup)
📱 Fully Responsive Design


Tech Stack

Frontend:

React.js
HTML5, CSS3, JavaScript

Backend:

Node.js
Express.js

Database:

MongoDB

Other Tools & Technologies:

REST APIs
JWT Authentication
Git & GitHub

# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Babel](https://babeljs.io/) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh
